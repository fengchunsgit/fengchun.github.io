Emmet常用命令

Posted on 2017-03-03 | In 笔记 |

使用Emmet插件能够大幅度提升编写HTML和CSS的效率，许多IDE也内置了Emmet，官方网站 emmet.io

html和css缩写

Emmet基本用法

  • 子元素 >
  • 同级元素 +
  • 爬升 ^
  • 重复 *
  • 调整组合顺序 ()

div>ul>li*3

div+div+p

div>div^p

(header>div>ul>li*4)+footer

Emmet属性

  • id #
  • class .
  • 自定义属性 []
  • 变量 $
  • 文字 {}
  • 无意义文字 lorem

div#container.container[data-value]

ul>li.item$*5

ul>li.item-$*5

a{click me}

lorem10 创造10个无意义文字

注意事项

  • tab键进行展开，跳转
  • 光标一定要在最后
  • 中间一定不能有空格
